# TrikHub Integration

Extends Home Assistant Assist with third-party AI skills (Triks).

## Features

- LangGraph-based conversation agent
- Multiple LLM providers (OpenAI, Anthropic, Ollama)
- Install Triks from the TrikHub registry
- HA services for automation integration

## Requirements

- TrikHub Server addon or external server
- LLM API key or Ollama instance
