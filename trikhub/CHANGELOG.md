# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-18

### Added

- Initial release of TrikHub Server addon
- REST API for Trik management (install, uninstall, list, reload)
- REST API for Trik execution
- REST API for tool discovery
- Swagger UI for interactive API documentation
- Persistent storage for installed Triks
- Configuration options for auth, registry URL, and logging
- Multi-architecture support (amd64, aarch64, armv7)
